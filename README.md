⚡ Terminal de Estrategia Deportiva (Surebet Finder COL)
Este proyecto es una herramienta avanzada desarrollada en Google Apps Script que convierte una hoja de Google Sheets en una terminal de análisis financiero para apuestas deportivas, optimizada para el mercado colombiano (Liga BetPlay, Libertadores, etc.).

🚀 Características principales
Dashboard Dark Mode: Interfaz profesional inspirada en terminales de trading.

Radar de Cuotas Dinámico: Conexión vía API para obtener cuotas en tiempo real (Local, Empate, Visitante).

Calculadora de Arbitraje (Surebets): Algoritmo que calcula la distribución exacta del capital para minimizar riesgos.

Auto-Relleno Inteligente: Lista desplegable que importa automáticamente los datos del partido seleccionado.

Optimizado para Colombia: Resultados redondeados a múltiplos de $50 COP, ideal para plataformas como Wplay o BetPlay.

🛠️ Instalación y Configuración
Crea una hoja de cálculo en Google Sheets.

Crea dos pestañas: PANEL_CONTROL y MOTOR_CUOTAS.

Ve a Extensiones > Apps Script.

Copia el código de codigo.gs (el archivo que subirás) y pégalo allí.

Importante: Debes obtener tu propia API KEY en The Odds API y colocarla en la constante API_KEY.

Guarda y refresca la hoja. Aparecerá el menú 🚀 OPERACIONES.

💻 Tecnologías usadas
Google Apps Script (JavaScript)

Google Sheets API

The Odds API (Data sourcing)
